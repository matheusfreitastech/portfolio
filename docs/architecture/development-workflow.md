# Development Workflow

This project is developed using an agile-inspired workflow.

Each sprint follows the same documentation structure:

- 🎯 Goal
- 📦 Planned Deliverables
- ✅ Acceptance Criteria
- 🏗️ Architecture Decisions
- 🧠 Technical Concepts
- 📚 Learning Outcomes
- 📸 Sprint Gallery
- 📈 Retrospective

This workflow documents not only the implementation itself, but also the reasoning and learning process behind each sprint.
